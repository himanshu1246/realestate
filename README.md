# Vrundavan Building Website

A simple static website scaffold for a building or construction business.

## Files
- `index.html` — landing page
- `styles.css` — page styling
- `script.js` — mobile menu and contact form behavior

## Usage
Open `index.html` in a browser to view the site.

To edit the website, update `index.html`, `styles.css`, and `script.js`.
